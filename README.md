# Paper Trading Dashboard (public)

Static dark-mode dashboard for the autonomous paper trading bot.

- Data source: `data/status.json` (generated by engine every ~5 minutes)
- Tech: HTML/CSS/JS + Chart.js
- Auto-update: systemd timer commits and pushes every 5 minutes with new data

Open `index.html` locally or host on any static host. `status.json` shape:
```
{
  "as_of": "2026-02-27T20:05:08Z",
  "prices": { "BTCUSDT": 61000.0, ... },
  "positions": [ { "symbol": "BTCUSDT", "qty": 0.1, "entry_price": 60000, "entry_time": 1700000000 } ],
  "trades": [ { "symbol": "BTCUSDT", "side": "BUY", "qty": 0.1, "price": 60000, "ts": 1700000000 } ],
  "equity": [ { "ts": 1700000000, "equity": 20000, "cash": 20000 } ]
}
```
